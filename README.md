# Sistema de Xadrez em Java

Um jogo completo de xadrez desenvolvido em Java utilizando **Programação Orientada a Objetos (POO)**. Projeto desenvolvido como parte do curso de Java e OO do professor Nelio Alves.

## Funcionalidades

- Regras completas de movimento de todas as peças (Peão, Torre, Bispo, Cavalo, Rainha e Rei)
- Detecção de movimentos especiais:
  - Roque (pequeno e grande)
  - En passant
  - Promoção de peão
- Validação de movimentos inválidos e xeque
- Interface no console (visualização do tabuleiro colorida)
- Contagem de movimentos e turnos
- Sistema de peças capturadas

## 🛠️ Tecnologias Utilizadas

- **Java** (JDK 17 ou superior)
- Programação Orientada a Objetos:
  - Herança
  - Polimorfismo
  - Encapsulamento
  - Classes abstratas e Interfaces
- Collections (List, etc.)
